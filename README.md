## Node React Starter ##

[![Build Status][travis-image]][travis-url] [![Coverage Status][coveralls-image]][coveralls-url]


An opinionated boilerplate for full stack node development with react and flux.

```
nvm install v0.12.7
nvm alias default v0.12.7

npm install
npm install -g gulp

gulp build

npm start
```

TESTING
=========
```
npm install -g mocha
./test.sh
```
